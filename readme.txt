Customize the currency symbol and price label for WooCommerce products.

== Description ==

The Custom Price Label plugin allows you to customize the currency symbol and price label displayed for WooCommerce products. Easily replace the default currency symbol with a custom one and modify the price label to suit your needs. Whether you want to display prices in a different currency format or add additional information to the price label, this plugin provides you with the flexibility to tailor the display according to your preferences. With intuitive settings and seamless integration with WooCommerce, you can enhance the presentation of your products and create a unique shopping experience for your customers.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= How do I do XYZ? =
= Can I use this plugin to display prices in multiple currencies?
Currently, the plugin allows you to customize the currency symbol and price label for products, but it doesn't support automatic conversion or display of prices in multiple currencies. However, you can manually set the currency symbol to match your desired currency format.

= Will this plugin affect the functionality of my existing WooCommerce setup?
No, this plugin is designed to work seamlessly with WooCommerce without affecting its core functionality. It simply provides options to customize the currency symbol and price label, giving you more control over how product prices are displayed to your customers.

= Can I revert back to the default currency symbol if needed?
Yes, you can easily revert back to the default currency symbol by adjusting the plugin settings. Simply select the "Use Default Label" option for the price label, and the default currency symbol configured in your WooCommerce settings will be applied.
